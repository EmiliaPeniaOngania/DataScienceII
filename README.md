# DataScienceII

# Modelo de Riesgo Crediticio - BCRA

## Descripción
Análisis exploratorio del riesgo crediticio de entidades financieras en Argentina utilizando datos del BCRA.

## Funcionalidades
- Selección dinámica de banco
- Cálculo de cartera irregular
- Indicadores financieros
- Visualización de evolución y comparación

## Datos
Fuente: BCRA

https://www.bcra.gob.ar/informacion-sobre-entidades-financieras/

## Archivos:
- Carga_datos_MySql.ipynb: se utiliza para cargar los archivos planos que se bajan desde el BCRA a MySql, el código controla las tablas que existen y
  busca en todas las carpetas el archivo que contiene la informacion completa de todas las entidades financieras
- Informacion_BCRA: contine la base de MySql con la Informacion del BCRA cargada
- PrimerEntregaDataScience.ipynb: contiene los analisis realizados sobre la información del BCRA asi como tb la conversion de datos, calculo de indicadores y demas acciones
  necesarias para el armado futuro del Modelo
- PRIMER ENTREGA - MODELO DATA SCIENCE: pdf con explicación del trabajo realiado
